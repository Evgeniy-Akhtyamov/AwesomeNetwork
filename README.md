# AwesomeNetwork
## Социальная сеть - ASP.NET Core MVC приложение

### Список функций соцсети:

* Регистрация
* Авторизация
* Страница пользователя с отображением фото, имени и других данных
* Изменение данных пользователя
* Добавление друзей
* Отправка сообщений друзьям
